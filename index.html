📘 VPN အသုံးပြုပုံ

အသစ်၀ယ်ယူသူများအတွက်

  1. VPN ဆော့ဝဲကို ကူးပါ။
  2. ဆော့ဝဲထဲ၀င်တဲ့ချိန် အင်တာနက် ခဏလိုပါတယ် ဖိုင်တွေပေါ်အောင်။
  3. ညာဘက်ထိပ်က ကြယ်လေးထဲမှာ Device ID ရှိပါတယ်။ Copy ယူပြီး Messenger မှာပို့ပေးပါ။
  4. Member ၀င်ပြီးရင် VPN ကို Exit နဲ့ ပြန်ထွက်ပါ။
  5. ပြန်၀င်တဲ့အချိန် အင်တာနက် ခဏရှိရမယ်၊ အဲ့တစ်ချိန် အလိုအလျောက် member ၀င်ပါပြီ။

သက်တမ်းတိုးသူများအတွက်

  1. ID ကို Copy ယူပြီး ပို့ပေးပါ။
  2. ကျနော်ဘက်က Date ပို့ပြီးရင် ညာဘက်ထိပ်က မျှားအစိမ်းကွေးလေးကို နှိပ်ပါ (အင်တာနက် ခဏလိုပါတယ်နော်)။
  3. ပြီးရင် Date 📅 ကို စစ်ကြည့်ပါ။
  4. Date 📅 ပြောင်းသွားရင် ရပါပြီ။

📌 မှတ်ချက်

အသစ်၀ယ်သူဖြစ်ဖြစ်၊ သက်တမ်းတိုးသူဖြစ်ဖြစ် နောက်ဆုံး Version မကူးထားရင် ပြန်ကူးပါ။ ကူးပြီးသွားရင်တော့ မလိုပါဘူး။
